服务器端需要为客户端生成source和secret，签名的时候会用到。

  3. 计算签名值 ：md5(username=xx&userpwd=yy&source=zz + md5(secret))

车国（C端）

请求：Form表单形式
响应：JSON形式

服务器端需要为客户端生成source和secret，签名的时候会用到。还需要指定client_id、client_secret、grant_type，登录时会用到。

登录

  • 参数：username、md5( userpwd )、client_id、client_secret、grant_type

  • 签名：

  1. 将source放入请求参数列表中
  2. 将请求参数列表按照key值的大小，升序排序
  3. 计算签名值 ：md5(username=xx&password=yy&source=zz + md5(secret))
  • 请求(密码需要用md5取哈希值)
client_id=1cfea82ba40a417f9d26c50b13174305&client_secret=52b2d96bac2b42b89c8732f482b947fb&grant_type=password&password=0ab44bd43d6b18fcd5cd928d6faab1b8&signature=039312d0b8c05748cf2ec53c8f339578&source=1cfea82ba40a417f9d26c50b13174305&username=15271061373
  • 响应：
{
    "access_token": "159120a8-2424-4ab5-ad54-9a8d8255536c",
    "token_type": "password",
